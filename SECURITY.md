# Security Policy

## Supported Versions

We currently support security updates for the latest `main` branch and the latest tagged release.

| Version | Supported |
|---------|-----------|
| Latest release | :white_check_mark: |
| main | :white_check_mark: |
| Older versions | :x: |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report vulnerabilities privately by emailing:

- **contact@gaeld.ch**

Include as much detail as possible:

- Affected component
- Steps to reproduce
- Potential impact
- Suggested remediation (if available)

## Response Timeline

We aim to:

- Acknowledge receipt within **72 hours**
- Provide an initial assessment within **7 days**
- Share remediation status regularly until resolution

## Disclosure Policy

- We investigate all valid reports confidentially.
- We request responsible disclosure and coordination before public release.
- Once fixed, we will publish a security advisory in release notes/changelog.

Thank you for helping keep Gäld and its users safe.
